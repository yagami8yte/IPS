# Default Cafe Image Setup Guide

## Purpose
This image will be displayed when no advertisement videos are available in the Assets/Ads folder.

## Requirements
- **Filename**: `DefaultCafe.jpg`
- **Location**: `C:\Users\jinho\DEV\IPS\Assets\Ads\DefaultCafe.jpg`
- **Recommended Resolution**: 1920x1080 or higher (landscape orientation)
- **Format**: JPG or JPEG
- **Aspect Ratio**: 16:9 (widescreen) or similar

## Where to Download Free High-Resolution Cafe Images

### Option 1: Unsplash (Recommended)
1. Visit: https://unsplash.com/s/photos/coffee-shop
2. Search for: "organic cafe interior" or "coffee shop"
3. Find a warm, inviting cafe image
4. Click on the image you like
5. Click "Download free" button
6. Rename the downloaded file to `DefaultCafe.jpg`
7. Move it to: `C:\Users\jinho\DEV\IPS\Assets\Ads\`

### Option 2: Pexels
1. Visit: https://www.pexels.com/search/coffee%20shop%20interior/
2. Browse and select a high-quality cafe image
3. Click "Free Download" and select "Original" size
4. Rename to `DefaultCafe.jpg`
5. Place in Assets/Ads folder

### Option 3: Pixabay
1. Visit: https://pixabay.com/images/search/cafe/
2. Filter by "Photos" and "Horizontal"
3. Download your chosen image
4. Rename to `DefaultCafe.jpg`
5. Place in Assets/Ads folder

## Recommended Search Keywords
- "modern coffee shop interior"
- "organic cafe"
- "cozy coffee shop"
- "minimalist cafe"
- "coffee shop aesthetic"
- "rustic cafe interior"

## What to Look For
- **Warm atmosphere**: Inviting, welcoming vibes
- **Good lighting**: Natural light or warm ambient lighting
- **Clean composition**: Not too cluttered
- **Brand alignment**: Matches your COFFEEBOTIC brand aesthetic
- **High resolution**: At least 1920x1080 pixels

## Installation Steps
1. Download your chosen image from one of the free stock photo sites above
2. Rename the file to: `DefaultCafe.jpg`
3. Copy the file to: `C:\Users\jinho\DEV\IPS\Assets\Ads\`
4. Rebuild and run the application
5. When no videos are available, the cafe image will automatically display

## Fallback Behavior
If `DefaultCafe.jpg` is not found, the application will show:
- A play icon
- "Welcome to COFFEEBOTIC" text
- "Tap to Start Your Order" message
